# datainsightR

**Quick Data Cleaning and Exploratory Insights in R**

---

## Overview

`datainsightR` is a lightweight R package designed to simplify the process of exploring and cleaning data.  
It provides a set of helper functions to:

- Summarize missing values  
- Detect numeric outliers  
- Clean column names into a consistent format  
- Generate quick descriptive statistics  

The package is intended for analysts and data scientists who want fast, readable insights without writing repetitive code.
